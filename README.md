# COVID-19 Vaccination Management System 💉

A professional database-driven web application to manage and track vaccination doses.

## 🚀 Live Demo
You can access the live project here: [mubashirsiddiqui.infinityfreeapp.com](http://mubashirsiddiqui.infinityfreeapp.com/)

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Backend:** PHP
- **Database:** MySQL
- **Tools:** VS Code, InfinityFree Hosting

## 📌 Features
- Secure Login System
- Real-time Database Records
- Responsive Design (Mobile-First)
